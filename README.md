# [gijs6.nl](https://gijs6.nl)

My personal site. I hope you like it. Please like it.

## Setup

```bash
python -m venv .venv
.venv/bin/pip install -r requirements.txt
npm install
```

## Serve/build

```bash
.venv/bin/python engine.py serve
.venv/bin/python engine.py build
```

## Using `bake`

If you have [bake](https://git.dupunkto.org/~meta/dotfiles/raw/master/bin/bake) installed in your `$PATH`, you can also use `bake setup`, `bake serve` and `bake build`.
